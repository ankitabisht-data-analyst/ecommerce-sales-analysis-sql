# Dataset

This folder contains all source CSV files for the E-Commerce Sales Analysis project.

## Files Included

| File | Description |

| customers.csv | Customer profiles with city and segment |
| inventory.csv | Current stock levels per product |
| orders.csv | All purchase transactions |
| order_items.csv | Line items per order |
| products.csv | Product catalogue with pricing |
| payments.csv | Payment transaction records |
| returns.csv | Returned orders with reasons |
